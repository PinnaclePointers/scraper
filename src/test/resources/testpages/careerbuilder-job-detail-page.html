Front End / HTML Developer

Job Description


Responsibilities: A Kforce client is seeking a Front End / HTML Developer in Los Angeles, California (CA). Our team is working on a few different projects, and it will be the Front-End Web Developer's job to convert Photoshop comps into pixel-perfect, functioning web pages. Each product has a short timeline and will consist of weekly sprint cycles, so iterations will need to be constant and efficient. The front-end developer will work on business sites and client sites.


Job Requirements

Requirements:

  • A solid knowledge of hand-coding HTML, JavaScript and CSS for web is required. Experience with ASP.Net, Visual Studio is a plus

  • 5+ years product development experience with HTML5, CSS3, and object-oriented JavaScript

  • Advanced experience with JQuery

  • Strong understanding of responsive design

  • Deep understanding of DOM and AJAX/JSON

  • Experience with version control

  • Experience with JavaScript frameworks (e.g. Backbone, EmberJS, Angular)

  • Familiarity with browser compatibility issues

  • Ability to multi-task effectively in a fast-paced deadline-driven environment

  • A passion for innovation and the latest web technologies

  • Ability to take creative direction and work independently

  • Excellent verbal and written communications skills required

  • Experience in working in an Agile environment

  • Great organizational skills, acute attention to detail, and high degree of accuracy



Optional Desired Skills:

  • Regular Expression wizardry

  • Understanding of XML/XSL transformations (including XPath queries)

  • C++ or Java Development Experience

  • Experience working with Visual Studio

  • SASS or LESS experience

  • Experience with bug tracking and time estimations (e.g. Jira/Greenhopper)

  • Experience in developing native Android and/or iOS apps
